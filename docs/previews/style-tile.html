03

Surface tiers

Three tiers. Rule: one tier per section. Tier 1 for quiet content, tier 2 for sectional cards, tier 3 reserved for hero / "wow" moments.

Tier 1

Flat matte.

Footer cards, contact form, low-frequency content.

.tier-flat · --surface-1

Tier 2

Frosted glass.

Service cards, pathway tabs, most sectional surfaces.

.tier-glass · backdrop-blur(20)

Tier 3

Lit glass.

Featured player, hero moments, the final CTA panel.

.tier-lit · radial-gradient + shadow
04

Motion

Four named easings. Three duration tiers (250 / 480 / 1200ms+). Never 300ms — it's the default-feeling tell. Hover any box.

--ease-out-expo · 480ms entrances
--ease-in-out-quint · 600ms state transitions
linear · 600ms marquees only
--ease-out-expo · 1400ms cinematic / hero
05

Interactives

Buttons, focus rings, micro-interactions. Hover lift = 1px, electric focus ring at 2px offset. No bounces, no rotations.

Gold ghost

Tab through the buttons to see the electric focus ring ( 0 0 0 4px var(--electric) with 2px page-floor offset).

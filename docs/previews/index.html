How to review

Open each preview · react · tell us what to change.

  1. Open every page in this list. Take 30-60s on each.
  2. Mark anything you want changed: a color, a component, a copy line, a layout.
  3. For the logo, pick one of the five concepts.
  4. For the command bar, decide: ship it / cut it / defer it.
  5. Reply with your decisions — I'll fold them into design.md and start Phase B implementation.
